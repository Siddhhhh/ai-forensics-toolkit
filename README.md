# 🧠 AI Forensics & Enhancement Toolkit (V2) #

An experimental **AI-powered toolkit** built with **Streamlit** that analyzes text and images to determine whether they were generated by artificial intelligence.  
The toolkit also includes **image enhancement utilities** and a **secure authentication system**.

---

## 🚀 Features  ##

- ✍️ **AI Text Authorship Detection**  
  Detect whether text was generated by AI using **DistilBERT embeddings + RandomForest classifier**

- 🖼️ **AI Image Authenticity Detection**  
  Identify AI-generated images using **ResNet deep feature extraction**

- 🪄 **Image Quality Enhancement**  
  Improve resolution and clarity of low-quality images

- 🧽 **Background Removal**  
  Automatically remove image backgrounds

- 🔐 **User Authentication System**  
  Login & registration system powered by **SQLite**

- 🧪 **Trial Mode**  
  Limited access to test toolkit features without logging in
