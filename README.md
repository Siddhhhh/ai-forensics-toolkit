# 🧠 AI Forensics & Enhancement Toolkit (V2)

A Streamlit-based toolkit for detecting AI-generated text and images while also providing image enhancement tools.

## Features
- AI Text Authorship Detection
- AI Image Authenticity Detection
- Image Quality Enhancement
- Background Removal
- User Authentication (SQLite)
- Trial Mode Access

## Tech Stack
- Python
- Streamlit
- PyTorch
- scikit-learn
- OpenCV
- SQLite

## Run the Project

Install dependencies:

--------
pip install -r requirements.txt
--------

Start the application:

--------
streamlit run app.py
--------
